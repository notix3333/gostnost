# University Report Guide

Source: `Методические указания к оформлению отчетов по ГОСТу.docx`.

This reference summarizes the attached department guide. Use it as the main local formatting reference after direct user and teacher requirements.

## Document Requirements

- File name format: `Surname_Group_DisciplineAbbreviation_WorkTypeNoWorkNumber`, for example `Ivanov_BBI-21-1_IKG_PR№1`.
- Page margins: top 2 cm, bottom 2 cm, left 3 cm, right 1.5 cm.
- Page numbering must be present. Numbering starts from the first page, but the visible number starts on the second page when the document has a content section.
- A page should not contain more than 25% empty space.

## Headings

- Font: Times New Roman.
- Size: 12 pt.
- Alignment: left for numbered headings.
- Style: bold.
- Indents: left 0 cm, right 0 cm, first line 1.25 cm.
- Spacing: before and after 24 pt, line spacing 1.5.
- Number headings automatically as a multilevel list.
- Number alignment: 1.25 cm from the left, text indent 1.25 cm, space after number.
- Lower-level headings must include previous-level numbers.
- Structural headings `ВВЕДЕНИЕ`, `ЗАКЛЮЧЕНИЕ`, `СОДЕРЖАНИЕ`, `СПИСОК ИСПОЛЬЗОВАННЫХ ИСТОЧНИКОВ`, `ПРИЛОЖЕНИЕ` are centered, uppercase, without a final period, without underline, and without first-line indent.
- Do not use generic headings such as `Ход работы` when a meaningful section title is possible.

## Main Text

- Font: Times New Roman.
- Size: 12 pt.
- Alignment: justified.
- Indents: left and right 0 cm, first line 1.25 cm.
- Spacing: before and after 0 pt, line spacing 1.5.
- State the goal before the main text.
- Do not insert theory unless it is necessary for the report.
- A section must not end with a figure, table, or list; add explanatory or concluding text after them.
- Follow the pattern `Text -> Figure`: explain the action or result first, then insert the figure.
- Use cross-references for figures and captions where possible.

## Figures And Screenshots

- Figure object: Times New Roman, 12 pt, centered, no first-line indent, line spacing 1.5.
- Caption: Times New Roman, 12 pt, centered, no first-line indent, line spacing 1.5.
- A figure must not exceed page margins.
- Use an en dash in figure captions: `Рисунок 3 – Пример наименования рисунка`.
- Every figure must have a cross-reference in the text.
- Screenshots must not be cropped; preserve the full image and its proportions.

## Tables

- Table text: Times New Roman, 10 pt.
- Table text alignment: left.
- Table text spacing: before and after 0 pt, single line spacing.
- Table width: 100%, centered.
- Table title: Times New Roman, 12 pt, left aligned, no first-line indent, line spacing 1.5.
- If a table continues on the next page, use `Продолжение таблицы 1` and repeat the header with numbered columns.
- Tables must not contain an unnecessary ordinal-number column.
- Tables must not use cell fill.
- Tables must have a header row.
- Tables must not be inserted as images.
- After a table, include text with spacing before 6 pt.

## Lists

- Font: Times New Roman, 12 pt.
- Alignment: justified.
- Indents: left and right 0 cm, no first-line indent.
- Spacing: before and after 0 pt, line spacing 1.5.
- Marker position: 0 cm; text indent: 1.25 cm; symbol after number: tab.
- Use a long dash marker where the guide requires it.
- Nested enumeration may use Russian lowercase letters with a closing parenthesis, excluding `ё`, `э`, `й`, `о`, `ч`, `ъ`, `ы`, `ь`.
- If the number of items is fixed, Arabic numerals with a closing parenthesis are allowed.

## Title Page

- Use the attached title page form as a fixed template.
- Preserve the original formatting when inserting or filling the title page.
- The guide recommends protecting the title page through a Word content control named and tagged `Титульный лист`, with the `Титульный лист` style attached.

## Formulas

- Put formulas on separate lines.
- Explain symbols directly below the formula in the same order as in the formula.
- Start the first explanation line with `где` without a colon.
- Do not insert formulas as screenshots or images.
- Use Word's formula tool where possible.
- Center formulas and number them consecutively in parentheses at the right edge of the line.

## Appendices

- Refer to every appendix in the report text.
- Place appendices in the order of text references.
- Start every appendix on a new page.
- Put `ПРИЛОЖЕНИЕ` at the top center.
- The appendix title starts with a capital letter, uses bold, is centered on a separate line, and has no final period.
- Mark appendices by uppercase Russian letters starting with `А`, excluding `Ё`, `З`, `Й`, `О`, `Ч`, `Ъ`, `Ы`, `Ь`.
- Include appendices in the content section when the report has one.
- Number appendix figures and tables with the appendix letter.

## Bibliography

- The guide references GOST 7.32-2017 for the list of sources.
- In-text source references must be in square brackets.
- Sources should usually be no older than five years unless the source type or assignment justifies it.
- The guide uses both `СПИСОК ИСПОЛЬЗОВАННЫХ ИСТОЧНИКОВ` and `СПИСОК ИСПОЛЬЗУЕМЫХ ИСТОЧНИКОВ`; treat this as a wording conflict and ask the user or follow the teacher's wording.

## Content Section

- Replace `Оглавление` with `СОДЕРЖАНИЕ`.
- Use uppercase for `СОДЕРЖАНИЕ`, `ВВЕДЕНИЕ`, `ЗАКЛЮЧЕНИЕ`, bibliography heading, and `ПРИЛОЖЕНИЕ`.
- `СОДЕРЖАНИЕ`: Times New Roman, uppercase, bold, 12 pt, centered, before 0 pt, after 24 pt, line spacing 1.5.
- Content entries use Times New Roman, 12 pt.
