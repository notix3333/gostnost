# Title Page Rules

Source: `Приложение. Форма титульного листа.docx`.

## Protected Template

- Treat `assets/title-page-template.docx` as immutable.
- Do not redraw the title page.
- Do not rewrite static institutional text.
- Do not change fonts, alignment, spacing, tables, margins, or visual placement.
- Do not "improve" the title page layout.
- Fill only clearly variable fields.

## Clearly Variable Fields In The Attached Template

The attached title page visibly contains these fillable or context-dependent areas:

- Work type or title line, currently shown as `Реферат`.
- Discipline name after `по дисциплине`.
- Topic after `на тему:`.
- Direction of study and semester when the task requires it.
- Student full name under `Выполнил`.
- Teacher full name under `Проверил`.
- Student group.
- Submission date.
- Check date, only if the user or teacher asks to fill it.
- City and year in the bottom line.

## Fields Requiring Caution

Ask the user before changing:

- University name.
- Department name.
- Institute or faculty name.
- Direction code and name.
- Semester.
- Work type if the template says `Реферат` but the user asks for practical, laboratory, or course work.
- Any line that is not clearly a placeholder or blank field.

## Filling Rule

When filling the title page, replace only the content of the target variable field. Preserve the original paragraph, run, table, and page formatting as much as the editing method allows. If preserving formatting cannot be guaranteed, stop and ask the user before proceeding.
