# Formatting Checklist

Use this checklist before returning a report or review result.

## Source And Scope

- User requirements are applied.
- Teacher requirements are applied.
- University guide requirements are applied.
- GOST is used as the baseline.
- Conflicts between guide and GOST are reported.
- Missing data is not invented.
- Placeholders are visible where data is missing.

## Title Page

- Protected title page template is used.
- Title page structure and visual formatting are unchanged.
- Only clear variable fields are filled.
- Unclear title-page fields were clarified with the user.
- City and year are correct or marked with placeholders.

## Document Layout

- Margins: top 2 cm, bottom 2 cm, left 3 cm, right 1.5 cm unless overridden.
- Page numbering exists when required.
- Visible page number starts from the second page when the report has content.
- No page has excessive empty space.
- File name follows the guide if the final filename is part of the task.

## Text And Headings

- Main text uses Times New Roman, 12 pt, justified, first-line indent 1.25 cm, line spacing 1.5.
- Headings use the guide's numbering and spacing rules.
- Structural headings are uppercase, centered, and without a final period.
- `СОДЕРЖАНИЕ` is used instead of `Оглавление`.
- The report is written in formal academic Russian from the student's perspective.
- Conversational phrases are removed.

## Structure

- Goal is present.
- Tasks are present when needed.
- Main sections match the assignment.
- Conclusion is present.
- Goal and conclusion correspond to each other.
- Sections do not end with a figure, table, or list.

## Figures And Screenshots

- Every figure has a caption.
- Figure captions use `Рисунок N – Название`.
- Every figure has a text reference.
- Screenshots are not cropped.
- Screenshots keep their original proportions.
- Screenshots are readable after scaling.
- Oversized screenshots are moved to a new page or appendix.

## Tables

- Every table has a title.
- Every table has a text reference.
- Tables are real tables, not images.
- Tables contain header rows.
- Table text uses 10 pt when following the guide.
- Continued tables are marked correctly.
- No unnecessary ordinal-number column is added.
- No cell fill is used unless the teacher requires it.

## Formulas

- Formulas are inserted as formulas, not images.
- Formulas are centered and numbered.
- Symbol explanations begin with `где` without a colon.
- Formula numbering is consistent.

## Sources And Appendices

- Bibliography heading matches teacher or guide requirements.
- Every listed source is referenced in text when required.
- In-text source references use square brackets.
- Appendices are referenced in the text.
- Appendices start on new pages and are included in content when required.
