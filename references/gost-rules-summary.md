# GOST Rules Summary

This file is a local working summary for student reports. It must not replace the attached university guide or the teacher's direct requirements.

## Baseline

- Treat GOST as the baseline standard for formal report structure, bibliographic discipline, figure and table captions, formula numbering, references, appendices, and page layout.
- When the university guide is stricter or more specific than the baseline GOST practice, follow the guide after reporting the difference if it matters to the user.
- Do not download external copies of GOST or examples unless the user explicitly asks.

## Report Elements

- Use a title page, content section, introduction or goal statement, main body, conclusion, bibliography when sources are used, and appendices when needed.
- Use formal Russian academic prose from the student's perspective.
- Do not invent source data, calculations, experiment results, names, groups, variants, or teacher information.
- Use square-bracket placeholders for missing data.

## Figures And Screenshots

- Every figure must have a caption and a text reference before or near the figure.
- Use the caption form `Рисунок 1 – Название рисунка`.
- Screenshots must be inserted completely, without cropping or loss of content.
- Preserve screenshot proportions and readability.

## Tables

- Every table must have a title and a reference in the text.
- Tables must be real tables, not screenshots.
- A table must contain a header row.
- Do not add an unnecessary ordinal-number column unless the assignment requires it.

## Formulas

- Formulas are separate from the main text.
- Do not insert formulas as images.
- Number formulas consecutively with Arabic numerals in parentheses, aligned to the right.
- Explain symbols immediately below the formula, starting with `где` without a colon.

## Bibliography

- Use the section heading required by the guide or teacher.
- Give in-text references in square brackets, for example `[1]`.
- Prefer sources from the last five years when the guide requires it, unless the source is a standard, law, classic method, or teacher-required material.
