# Placeholders

Use placeholders in square brackets when required data is missing. Do not invent data.

| Placeholder | Replace With |
| --- | --- |
| `[ФИО СТУДЕНТА]` | Student full name exactly as it should appear on the report. |
| `[ГРУППА]` | Student group number or code. |
| `[ДИСЦИПЛИНА]` | Discipline name from the assignment or teacher. |
| `[ТЕМА РАБОТЫ]` | Exact report topic. |
| `[ТИП РАБОТЫ]` | Practical, laboratory, course work, abstract, or another required type. |
| `[ФИО ПРЕПОДАВАТЕЛЯ]` | Teacher full name and title if required. |
| `[КАФЕДРА]` | Department name. |
| `[ВУЗ]` | University name. |
| `[ГОРОД]` | City on the title page. |
| `[ГОД]` | Report year. |
| `[ВАРИАНТ]` | Assignment variant. |
| `[ДОБАВИТЬ СКРИНШОТ]` | Place where the user must provide a screenshot. |
| `[ДОБАВИТЬ РАСЧЕТЫ]` | Place where actual calculations or results must be inserted. |
| `[ДОБАВИТЬ ВЫВОД]` | Place where a conclusion based on actual results must be inserted. |

If a placeholder remains in the final report, mention it in the review output as data to clarify.
